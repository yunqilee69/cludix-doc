# Jellyfin 安装与配置指南

Jellyfin 是一款免费、开源的媒体服务器软件，可以用于存储、管理和播放您的视频、音乐等媒体内容。

## 服务端安装

详细的 Docker 部署步骤，请参考：[Docker 部署 Jellyfin](/dev-ops/docker/jellyfin/)

服务启动后，访问 http://localhost:8096 进行初始化配置。

## 电影目录规范

### 目录结构

推荐使用目录划分，每个目录代表一部电影，便于整理和管理：

```
/media/
  ├── 肖申克的救赎 (1994)/
  │   ├── 肖申克的救赎 (1994).mkv
  │   ├── 肖申克的救赎 (1994).zh-CN.srt
  │   └── 肖申克的救赎 (1994).en.srt
  ├── 教父 (1972)/
  │   ├── 教父 (1972).mkv
  │   └── 教父 (1972).zh-CN.srt
  └── 星球大战 (1977)/
      ├── 星球大战 (1977).mkv
      └── ...
```

### 文件命名规范

- **目录名称**：推荐使用 `电影名称 (年份)` 格式
- **视频文件**：支持 `.mkv`、`.mp4`、`.avi`、`.mov` 等常见视频格式
- **年份**：可选，但推荐添加以便 Jellyfin 准确识别电影信息

## 字幕文件配置

### 字幕文件命名

字幕文件应与视频文件同名，添加语言后缀：

```
/movies/肖申克的救赎 (1994)/
  ├── 肖申克的救赎 (1994).mkv
  ├── 肖申克的救赎 (1994).zh-CN.srt
  └── 肖申克的救赎 (1994).en.srt
```

### 常见语言代码

- `zh-CN`：简体中文
- `zh-TW`：繁体中文
- `en`：英语
- `ja`：日语
- `ko`：韩语

### 字幕格式

支持 `.srt`、`.ass`、`.vtt` 等常见字幕格式。

:::tip
Jellyfin 会自动识别同目录下的字幕文件，用户在播放时可以通过播放器控制面板切换字幕。
:::

## 客户端安装（macOS）

### 下载地址

访问 Jellyfin 官方下载页面：https://jellyfin.org/downloads/

选择 **macOS** 平台，下载 `.dmg` 安装包。

### 安装步骤

1. 下载完成后，双击打开 `.dmg` 文件
2. 将 Jellyfin 应用拖拽到 Applications 文件夹
3. 从 Launchpad 或应用程序文件夹启动 Jellyfin

### 连接服务端

首次启动后，输入服务器地址：
- 如果在同一网络：`http://服务器IP:8096`
- 如果在本地测试：`http://localhost:8096`

:::warning
确保 macOS 设备与服务端在同一网络内，或服务端已配置端口转发以便外网访问。
:::
