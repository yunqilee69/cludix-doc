1. 先确认 APK 来自可信的官方发布渠道。
2. 按照文档开启 USB 调试并完成电脑 RSA 授权后，再执行 ADB 安装。
3. Codex 中缺少 Node.js 或 Java 命令时，按 [Codex 中 Node 与 Java 命令找不到的解决方法](./Codex中Node与Java命令找不到) 检查启动环境。
