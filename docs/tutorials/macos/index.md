## 使用建议

1. 首次配置开发机时，先安装 [Homebrew](./homebrew) 和 Java 环境。
2. 开发 HarmonyOS 时，再按照 [HarmonyOS 命令行开发环境](./harmonyos) 配置工具、SDK 和真机调试流程。
3. 开发 Android 应用时，阅读 [macOS 配置 Android 最小开发环境](./macOS配置Android开发环境)。
