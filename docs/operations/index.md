# 运维文档

本目录记录服务器运维和应用部署的相关文档，按工作流分为6个部分：

## 📚 文档分类

- [Basics](basics/debian-base-config/) - 系统配置、Docker、K8s等基础环境搭建
- [Docker](docker/) - 单应用 Compose + 共享网络（app-net）的统一部署规范
- [Kubernetes](k8s/install-k8s/) - 集群安装、Helm Chart 打包分享、镜像加速、NFS与Rancher部署
- [Network](network/frp-nat/) - 网络配置、内网穿透、SSH等工具
- [Services](services/jellyfin/) - Jellyfin、Milvus等应用服务部署
- [CI/CD](cicd/jenkins/) - Jenkins、Webhook等自动化部署工具
