# Mac mini 设置常开不休眠，但允许屏幕自动熄灭

本文档用于配置 `Mac mini` 长期开机运行，避免主机自动休眠，同时保留屏幕在无人操作时自动熄灭的能力。这个配置适合家用服务器、下载机、影音主机和远程访问场景。

## 适用场景

- `Mac mini` 需要持续联网或被远程访问
- 主机不能自动休眠
- 显示器可以在空闲后自动熄灭，以减少亮屏时间

## 配置目标

完成后应满足以下效果：

- `Mac mini` 不会因为空闲而自动进入睡眠
- 屏幕可以按照设定时间自动关闭
- 手动关机仍然可用，本配置不影响正常重启或关机操作

## 图形界面设置

### 1. 设置屏幕自动熄灭时间

打开：苹果菜单 > `系统设置` > `锁定屏幕`

找到以下选项：

- `使用电源适配器供电且不活跃时关闭显示器`

建议设置为以下任意一种：

- `10 分钟`
- `15 分钟`
- `30 分钟`

这一步的作用是：机器保持运行，但在无人操作一段时间后自动关闭显示器输出。

### 2. 阻止主机在屏幕熄灭后自动休眠

打开：苹果菜单 > `系统设置` > `能源`

开启以下选项：

- `显示器关闭时，防止自动进入睡眠`

这是最关键的一项。开启后，即使显示器已经自动熄灭，`Mac mini` 本机仍会继续运行，不会因为空闲而进入睡眠。

### 3. 可选：允许网络访问时自动响应

如果这台 `Mac mini` 还承担文件共享、远程桌面或其他局域网服务，可以按需开启：

- `唤醒以供网络访问`

如果只是本地常开使用，这一项可以保持默认。

## 结果检查

配置完成后，可以按下面方式简单验证：

1. 保持机器接通电源
2. 停止操作电脑，等待显示器按设定时间自动熄灭
3. 过几分钟后通过以下任一方式确认主机仍在运行：
   - 使用键盘或鼠标唤醒屏幕
   - 通过局域网 `SSH`、远程桌面或共享服务访问
   - 观察后台任务是否持续执行

如果屏幕熄灭后仍能正常访问机器，说明设置生效。

## 命令行检查（可选）

终端执行：

```bash
pmset -g
```

重点关注当前电源策略是否符合以下思路：

- 显示器允许在空闲后关闭
- 系统不会因为显示器关闭而自动进入睡眠

如果你只是按图形界面完成设置，一般不需要手动修改 `pmset`。

## 常见问题

### 屏幕熄灭后远程连不上机器

先确认以下几点：

- `系统设置` > `能源` 中已经开启 `显示器关闭时，防止自动进入睡眠`
- 机器始终连接电源适配器
- 没有额外使用第三方省电工具覆盖系统设置

### 机器会不会自动关机

正常情况下，macOS 不会因为空闲而自动关机。大多数场景下需要处理的是“自动休眠”，不是“自动关机”。

如果这台机器以前配置过定时关机、定时重启或第三方节能策略，需要额外检查并取消这些设置。

## 推荐配置

对于常年插电运行的 `Mac mini`，推荐使用下面这组设置：

- `锁定屏幕` > `使用电源适配器供电且不活跃时关闭显示器`：`15 分钟`
- `能源` > `显示器关闭时，防止自动进入睡眠`：`开启`
- `能源` > `唤醒以供网络访问`：按需开启

这样可以保证主机持续在线，同时避免显示器长时间常亮。
